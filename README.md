## Installation

1. Install project dependencies:

#### `yarn install`

2. Set InstaPy database path at src/services/server.js

#### `instapyRootFolder = '/path/to/InstaPy'`

3. Start the project

#### `yarn start`

4. http://localhost:3000

5. Log will start to show up when you run your InstaPy instance
